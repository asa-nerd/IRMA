# IRMA repositories

![IRMA-host.jpg](Documentation/IRMA-host.jpg)

This repository includes software, plans, documentation, and experimental data related to the measuring apparatus IRMA (Interactive Real-time Measurement of Attention). The system was developed for the laboratory concerts ob the GAPPP project and was used to record and analyze audience responses. GAPPP was a research project (led by Marko Ciciliani) at the University of Music and Performing Arts Graz/Austria from 2017 to 2021. http://gappp.net

## Modules

The measuring apparatus IRMA is designed in modules. 

![IRMA-modules.svg](Documentation/IRMA-modules.svg)

Each module has its own repository on Github. 

### 1 IRMA Interface

Code and documentation of IRMA's interface devices.  

### 2 IRMA Host

Contains documentation of the host app. 

### 3 IRMA Analysis

Provides documentation of the analysis app.

### 4 IRMA Case and Plans

Documentation and plans for the cases of the IRMA handheld devices. 

### 5 Raw Data

A repository providing the data collected with IRMA during the laboratory concerts of the GAPPP project.
